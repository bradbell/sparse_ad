name
Is the name for the matrix.

pattern
Is the sparsity pattern that is printed. In this case the row and column indices corresponding to possibly non-zero values are printed.

matrix
Is the sparse matrix that is printed. In this case the possibly non-zero values are printed along with the corresponding row and column indices.
