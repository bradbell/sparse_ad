csv_summary_py¶

\[\newcommand{\B}[1]{ {\bf #1} } \newcommand{\R}[1]{ {\rm #1} }\]

Summarize A CSV File’s Speed Results¶

  • Syntax

  • csv_file

  • problem

  • setup

  • Example Output

Syntax¶

bin/csv_summary.py csv_file problem setup

csv_file¶

This is name of a csv file created buy run_speed.sh. It must begin with csv/; i.e., it must be in the csv directory. It must end with .csv.

problem¶

This is the problem we are summarizing the results for and must be one of the following: dficfj, dierfj, deptfg, dgl1fg.

setup¶

This must be true or false and indicates in the :ref:speed.--setup time is included for this summary.
