Working Directory¶

All of the scripts below must be executed from the git root directory for this repository; i.e, the directory that contains the .gitignore file.

Children¶

  1. get_all_sh: Local install of Packages Required by sparse_ad

  2. wget_minpack2_sh: Get a Minpack2 File From Argonne National Laboratory

  3. f2c_sh: Create a C Version of a Fortran File

  4. c2hpp_py: Create a C++ Template Version of an f2c File

  5. csv2tex_py: Extract Latex Tables From Speed Test Csv File

  6. run_speed_sh: Run All the Jacobian and Hessian Speed Tests
