wget_minpack2_sh¶

\[\newcommand{\B}[1]{ {\bf #1} } \newcommand{\R}[1]{ {\rm #1} }\]

Get a Minpack2 File From Argonne National Laboratory¶

  • Syntax

  • Working Directory

  • file

  • Result

  • Example

Syntax¶

bin/wget_minpack2.sh minpack2 / file

Working Directory¶

see working_directory.

file¶

The file part of the command line argument minpack2 / file is a valid file location relative to the Argonne National Laboratory copy of minpack2; i.e.,

        ftp.mcs.anl.gov/pub/MINPACK-2 file

is the a valid file location.

Result¶

The file is copied to the location minpack2 / file relative to the current working directory (git root directory).

Example¶

If one is in the git root directory, the command

        bin/wget_minpack2.sh minpack2/tprobs.92/dficfj.f

will overwrite the git repository copy of dficfj.f . The result file should be the same which you can check using

        git diff minpack2/tprobs.92/dficfj.f
